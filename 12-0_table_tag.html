<테이블 태그들의 속성>

속성 값 설명
align left, center, right 테이블을 어디에 정렬할 것인지 지정
bgcolor 색값 테이블의 배경색 지정
background 배경 그림 테이블의 배경 그림 지정
border 두께 테이블 테두리의 두께 지정
bordercolor 색값 테이블 테두리의 색상 지정
cellspacing 간격 셀과 셀의 간격 지정
cellpadding 여백 셀 안의 내용과 여백 지정
height 크기, 비율 픽셀로 입력할 때 절댓값, % 단위로 입력 시 웹브라우저 크기에 대한 상대값
width 크기, 비율 픽셀로 지정할 때 절댓값, % 단위로 입력 시 웹브라우저 크기에 대한 상대값
